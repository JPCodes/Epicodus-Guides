Atom Pair Setup - (Open Terminal) - touch .pairs - atom .pairs (create pairs and email sections) Solo Git Setup - git config --global user.name "Firstname Lastname" - git config --global user.email email@email.com - git about (displays current user(s)) Git Guide for Pairing - Git init (starts tracking directory) - Git pair jr jp (adds authors to the git project) - Git add . (adds all files to watchlist) - Git status (to confirm directory is watched) - Git-pair-commit -m “Comment goes here” (to commit as a pair) - (Create Repository) - Git remote add jp url.com/ (Linking to repository, per person) - Git push jr master (pushes files to github, per person) - Git status (to confirm everything is up to date) Git Commands for Soloing - git add . - git commit -m "Comment goes here" (to commit solo) - git remote add origin https://github.com/user/repo.git - git push origin master Git Branching - git branch gh-pages (gh-pages can be any name) - git checkout gh-pages (switches to branch) - git push jp gh-pages (per person, pushes to branch) Git Delete a Branch - git branch (displays branches; master and other branches should display) - git branch -D gh-pages (deletes gh-pages branch) - git branch (displays branches; gh-pages should now be gone) Git Merging - git checkout master (switch to your master branch) - git branch (displays current branch) - git merge master gh-pages (this would merge the master branch with the gh-pages branch) Troubleshooting-related Commands - git branch (displays branches, star denotes current branch) - pwd (verify proper location) Erase Git Credentials (Error 403 Access Denied fix) - git credential-osxkeychain erase host=github.com protocol=https (press return twice) Wrong Git Repo - rm -rf .git (deletes git settings) - (Add remote as normal, etc)
